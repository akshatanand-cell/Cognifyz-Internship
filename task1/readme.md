# 🧩 Task 1 – HTML Structure & Basic Server Interaction

> **Cognifyz Technologies Internship** – Full Stack Development  
> **Focus:** Semantic HTML + Minimal Express Server

---

## 🎯 Objective

Create a basic HTML form (name, email, message) and set up a Node.js/Express server that receives the form data and displays a simple success page.

---

## 🚀 Features

- Semantic HTML5 form with proper labels and input types
- Express.js server running on port 3000
- POST endpoint `/submit` to handle form submission
- Success page showing submitted data
- Basic error handling for missing routes

---

## 🖥️ How to Run

cd task1
npm install
node server.js
Open http://localhost:3000 in your browser.
