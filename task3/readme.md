# 📱 Task 3 – Advanced CSS Styling & Responsive Design

> **Cognifyz Technologies Internship** – Full Stack Development  
> **Focus:** Mobile‑first design, Flexbox/Grid, media queries

---

## 🎯 Objective

Upgrade the form with modern CSS (Flexbox/Grid), make it fully responsive (mobile, tablet, desktop), and add hover/focus effects for better UX.

---

## 🚀 Features

- External CSS file (no inline styles)
- Responsive layout using CSS Grid and Flexbox
- Media queries for breakpoints: 480px, 768px, 1024px
- Smooth transitions and focus states
- Scalable typography (rem/em units)

---

## 🖥️ How to Run


cd task3
npm install
node server.js
Test on different device sizes using browser DevTools.
